# Image Assets Required

Please add the following images to complete your gym website:

## Required Images

### 1. Logo
**File:** `logo.png`
- **Location:** `/public/logo.png`
- **Size:** 512x512px (square)
- **Format:** PNG with transparent background
- **Description:** Your gym's official logo

### 2. Hero Background
**File:** `images/hero-bg.jpg`
- **Location:** `/public/images/hero-bg.jpg`
- **Size:** 1920x1080px (landscape)
- **Format:** JPG
- **Description:** Main hero section background - ideally showing your gym interior or equipment

### 3. About Section
**File:** `images/about-gym.jpg`
- **Location:** `/public/images/about-gym.jpg`
- **Size:** 1200x800px (landscape)
- **Format:** JPG
- **Description:** Image showing your gym's spacious interior or a training session

### 4. Gallery Images (6 images)
**Location:** `/public/images/gallery/`

1. `cardio-area.jpg` - Cardio equipment zone (1200x900px)
2. `strength-training.jpg` - Strength training area (1200x900px)
3. `free-weights.jpg` - Free weights section (1200x900px)
4. `gym-interior.jpg` - Overall gym interior (1200x900px)
5. `training-session.jpg` - People training (1200x900px)
6. `equipment.jpg` - Modern equipment closeup (1200x900px)

### 5. Open Graph Image (for social media sharing)
**File:** `og-image.jpg`
- **Location:** `/public/og-image.jpg`
- **Size:** 1200x630px
- **Format:** JPG
- **Description:** Image that will appear when your website is shared on social media

## Image Guidelines

- **Quality:** Use high-resolution images (min 1080p)
- **Format:** JPG for photos, PNG for logos/graphics
- **Compression:** Optimize images before uploading (use tools like TinyPNG)
- **Lighting:** Use well-lit, professional-looking photos
- **Subjects:** Show real gym equipment, actual facility, training sessions
- **Consistency:** Maintain similar color grading across all images

## Placeholder Images

Until you add your actual images, the website will show broken image placeholders.
You can use free stock photos from:
- [Unsplash](https://unsplash.com/s/photos/gym)
- [Pexels](https://www.pexels.com/search/fitness/)
- [Pixabay](https://pixabay.com/images/search/gym/)

## Creating the Directory

Run this command to create the images directory structure:

```bash
mkdir -p public/images/gallery
```

Then add your images to the respective folders.
